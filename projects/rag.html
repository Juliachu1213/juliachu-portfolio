1. Introduction / Problem

The hospitality industry produces massive amounts of unstructured user reviews on sites like Booking.com and Airbnb. While these reviews contain valuable insights, the scale makes it difficult for travelers to find relevant information efficiently. To address this, we built an intelligent hotel review question-answering chatbot that retrieves and summarizes hotel reviews using the Retrieval-Augmented Generation (RAG) framework. The system matches a user‚Äôs natural-language query (e.g., ‚ÄúWhich hotels are quiet and clean?‚Äù) with semantically similar hotel documents, then generates concise, context-aware recommendations.

2. Prior Work

We scraped and preprocessed over 200,000 English-language hotel reviews from Booking.com to create a structured corpus. Each hotel‚Äôs reviews were concatenated into a single document and embedded with SentenceTransformers stella_en_400M_v5) for semantic retrieval in subsequent tasks.

3. Model / Method

  • Corpus Preparation: Filtered reviews containing ‚Äúliked‚Äù and organized them per hotel.
  • Dense Retrieval: Encoded reviews and queries using all-MiniLM-L6-v2 and computed cosine similarity to rank hotels.
  • Prompt Construction: Summarized top-k documents via facebook/bart-large-cnn and inserted them into structured prompts.
  • Generation: Adopted DeepSeek-R1-Distill-Qwen-1.5B for fluent, structured answers listing pros, cons, and verdicts.
  • Interface: Deployed through Gradio for real-time Q&A interaction.

4. Results & Findings

The chatbot delivers concise, structured responses under 20 seconds end-to-end latency on CPU, demonstrating high semantic relevance with minimal hallucination. Qualitative evaluation confirmed accurate, well-organized results, though smaller LLMs struggled with vague queries and long-context reasoning.

5. Conclusion

This project demonstrates how RAG can bridge unstructured textual data and actionable insights for real-world travel decision support. Despite hardware limitations, the system effectively integrates retrieval, summarization, and generation to provide intelligent, trustworthy hotel recommendations.
