Introduction

This thesis addresses the limitations of the traditional CWMP (CPE WAN Management Protocol), which suffers from SOAP overhead, inefficient event notifications, and weak security mechanisms. By introducing the Web of Things (WoT) architecture, the research redefines CWMP‚Äôs SOAP-based RPC methods into RESTful APIs, improving interoperability and system modularity. WebSocket and Webhook are implemented to replace the CWMP Inform mechanism, significantly enhancing real-time communication.

Objectives

  • Enhance interoperability through RESTful API-based method mapping following WoT standards.
  • Improve event notification performance via WebSocket & Webhook.
  • Strengthen authentication and authorization using OAuth 2.0 and JSON Web Token (JWT).

System Design & Implementation

  • Thing Description (TD): Defines device and service resources in JSON-LD format for unified access.
  • OAuth 2.0 & JWT: Enables secure user authentication and resource authorization.
  • Database Layer (PostgreSQL): Stores device configurations, client credentials, and event logs.
  • Spring Boot Framework: Implements WoT scripting logic in Java, re-architecting the CWMP workflow.
  • API Testing: Conducted using Postman with GitHub OAuth integration for validation.

Conclusion

This thesis proposes a secure and efficient smart home gateway management system based on the Web of Things framework.
